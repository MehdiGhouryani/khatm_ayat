
A Telegram bot designed to assist users in counting and tracking the number of Quranic verses (ayat) recited during their study sessions.

## Features

- Count the number of verses recited
- Track and log recitation sessions
- View statistics and progress over time
- Receive reminders and motivational messages

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/MehdiGhouryani/khatm_ayat.git
````

2. Navigate into the project directory:

   ```bash
   cd khatm_ayat
   ```

3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Configure your bot:

   * Edit `config.py` with your Telegram bot token and any other necessary settings.

5. Run the bot:

   ```bash
   python main.py
   ```

## Usage

* Start the bot by sending `/start` in Telegram.
* Use the inline keyboard to navigate through different sections.
* Type `/help` to get a list of available commands.
